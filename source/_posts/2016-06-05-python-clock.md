## Python公告

Python 发布了一个网站 http://pythonclock.org/ 来开始 Python 2.7 版本的退休倒计时。
<!--more-->
该版本将在 2020年4月12日 退休。退休后将不再进行任何维护。正式退休将在 2020 年的 PyCon 上宣布。
那大家有秩序的逐步迁移到 Python 3 吧，不要挤，时间大把！
